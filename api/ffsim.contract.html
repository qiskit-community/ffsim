ffsim.contract¶

Functions for contracting tensors and constructing linear operators.

ffsim.contract.contract_diag_coulomb(vec, mat, norb, nelec, *, mat_alpha_beta=None, z_representation=False)[source]¶

Contract a diagonal Coulomb operator with a vector.

A diagonal Coulomb operator has the form

\[\sum_{i, j, \sigma, \tau} Z_{ij} n_{\sigma, i} n_{\tau, j} / 2\]

where \(n_{\sigma, i}\) denotes the number operator on orbital \(i\) with spin \(\sigma\) and \(Z\) is a real symmetric matrix If mat_alpha_beta is also given, then it is used in place of \(Z\) for the terms in the sum where the spins differ (\(\sigma \neq \tau\)).

Parameters:

  • vec (ndarray) – The state vector to be transformed.

  • mat (ndarray) – The real symmetric matrix \(Z\).

  • norb (int) – The number of spatial orbitals.

  • nelec (tuple[int, int]) – The number of alpha and beta electrons.

  • mat_alpha_beta (Optional[ndarray]) – A matrix of coefficients to use for interactions between orbitals with differing spin.

  • z_representation (bool) – Whether the input matrices are in the “Z” representation.

Return type:

ndarray

Returns:

The result of applying the diagonal Coulomb operator on the input state vector.

  • norb (int) – The number of spatial orbitals.

  • nelec (tuple[int, int]) – The number of alpha and beta electrons.

Returns:

The result of applying the linear combination of number operators on the input state vector.

ffsim.contract.diag_coulomb_linop(mat, norb, nelec, *, orbital_rotation=None, mat_alpha_beta=None, z_representation=False)[source]¶

Convert a (rotated) diagonal Coulomb matrix to a linear operator.

A rotated diagonal Coulomb operator has the form

\[\mathcal{U} (\sum_{i, j, \sigma, \tau} Z_{ij} n_{\sigma, i} n_{\tau, j} / 2) \mathcal{U}^\dagger\]

where \(n_{\sigma, i}\) denotes the number operator on orbital \(i\) with spin \(\sigma\), \(Z\) is a real symmetric matrix, and \(\mathcal{U}\) is an optional orbital rotation. If mat_alpha_beta is also given, then it is used in place of \(Z\) for the terms in the sum where the spins differ (\(\sigma \neq \tau\)).

Parameters:

  • mat (ndarray) – The real symmetric matrix \(Z\).

  • norb (int) – The number of spatial orbitals.

  • nelec (tuple[int, int]) – The number of alpha and beta electrons.

  • orbital_rotation (Optional[ndarray]) – A unitary matrix describing the optional orbital rotation.

  • mat_alpha_beta (Optional[ndarray]) – A matrix of coefficients to use for interactions between orbitals with differing spin.

  • z_representation (bool) – Whether the input matrices are in the “Z” representation.

Return type:

LinearOperator

Returns:

A LinearOperator that implements the action of the diagonal Coulomb operator.
