ffsim.contract.diag_coulomb module¶

Contract diagonal Coulomb operator.

ffsim.contract.diag_coulomb.contract_diag_coulomb(vec, mat, norb, nelec, *, mat_alpha_beta=None, z_representation=False, occupations_a=None, occupations_b=None, strings_a=None, strings_b=None)[source]¶

Contract a diagonal Coulomb operator with a vector.

A diagonal Coulomb operator has the form

\[\sum_{i, j, \sigma, \tau} Z_{ij} n_{i, \sigma} n_{j, \tau} / 2\]

where \(n_{i, \sigma}\) denotes the number operator on orbital \(i\) with spin \(\sigma\) and \(Z\) is a real symmetric matrix If mat_alpha_beta is also given, then it is used in place of \(Z\) for the terms in the sum where the spins differ (\(\sigma \neq \tau\)).

Parameters:

  • vec (ndarray) – The state vector to be transformed.

  • mat (ndarray) – The real symmetric matrix \(Z\).

  • norb (int) – The number of spatial orbitals.

  • nelec (tuple[int, int]) – The number of alpha and beta electrons.

  • mat_alpha_beta (Optional[ndarray]) – A matrix of coefficients to use for interactions between orbitals with differing spin.

  • z_representation (bool) – Whether the input matrices are in the “Z” representation.

  • occupations_a (Optional[ndarray]) – List of occupied orbital lists for alpha strings.

  • occupations_b (Optional[ndarray]) – List of occupied orbital lists for beta strings.

  • strings_a (Optional[ndarray]) – List of alpha strings.

  • strings_b (Optional[ndarray]) – List of beta strings.

Return type:

ndarray

Returns:

The result of applying the diagonal Coulomb operator on the input state vector.

ffsim.contract.diag_coulomb.diag_coulomb_linop(mat, norb, nelec, *, orbital_rotation=None, mat_alpha_beta=None, z_representation=False)[source]¶

Convert a (rotated) diagonal Coulomb matrix to a linear operator.

A rotated diagonal Coulomb operator has the form

\[\mathcal{U} (\sum_{i, j, \sigma, \tau} Z_{ij} n_{i, \sigma} n_{j, \tau} / 2) \mathcal{U}^\dagger\]

where \(n_{i, \sigma}\) denotes the number operator on orbital \(i\) with spin \(\sigma\), \(Z\) is a real symmetric matrix, and \(\mathcal{U}\) is an optional orbital rotation. If mat_alpha_beta is also given, then it is used in place of \(Z\) for the terms in the sum where the spins differ (\(\sigma \neq \tau\)).

Parameters:

  • mat (ndarray) – The real symmetric matrix \(Z\).

  • norb (int) – The number of spatial orbitals.

  • nelec (tuple[int, int]) – The number of alpha and beta electrons.

  • orbital_rotation (Optional[ndarray]) – A unitary matrix describing the optional orbital rotation.

  • mat_alpha_beta (Optional[ndarray]) – A matrix of coefficients to use for interactions between orbitals with differing spin.

  • z_representation (bool) – Whether the input matrices are in the “Z” representation.

Return type:

LinearOperator

Returns:

A LinearOperator that implements the action of the diagonal Coulomb operator.

ffsim.contract.hamiltonian module¶

Contract a molecular Hamiltonian.

ffsim.contract.hamiltonian.hamiltonian_linop(*, norb, nelec, one_body_tensor=None, two_body_tensor=None, constant=0.0)[source]¶

Convert a molecular Hamiltonian to a linear operator.

A molecular Hamiltonian has the form

\[H = \sum_{pq, \sigma} h_{pq} a^\dagger_{p, \sigma} a_{q, \sigma} + \frac12 \sum_{pqrs, \sigma} h_{pqrs, \sigma\tau} a^\dagger_{p, \sigma} a^\dagger_{r, \tau} a_{s, \tau} a_{q, \sigma}.\]

Here \(h_{pq}\) is called the one-body tensor and \(h_{pqrs}\) is called the two-body tensor.

Parameters:

  • one_body_tensor (Optional[ndarray]) – The one-body tensor.

  • two_body_tensor (Optional[ndarray]) – The two-body tensor.

  • norb (int) – The number of spatial orbitals.

  • nelec (tuple[int, int]) – The number of alpha and beta electrons.

  • constant (float) – The constant.

Return type:

LinearOperator

Returns:

A LinearOperator that implements the action of the Hamiltonian.
