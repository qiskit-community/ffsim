ffsim.qiskit¶

Code that uses Qiskit, e.g. for constructing quantum circuits.

class ffsim.qiskit.DiagCoulombEvolutionJW(mat, time, *, mat_alpha_beta=None, z_representation=False, label=None, validate=True, rtol=1e-05, atol=1e-08)[source]¶

Bases: Gate

Diagonal Coulomb evolution under the Jordan-Wigner transformation.

The diagonal Coulomb evolution gate has the unitary

\[\exp\left(-i t \sum_{\sigma, \tau, i, j} Z_{ij} n_{\sigma, i} n_{\tau, j} / 2\right)\]

where \(n_{\sigma, i}\) denotes the number operator on orbital \(i\) with spin \(\sigma\), \(Z\) is a real symmetric matrix. If mat_alpha_beta is also given, then it is used in place of \(Z\) for the terms in the sum where the spins differ (\(\sigma \neq \tau\)).

This gate assumes that qubits are ordered such that the first norb qubits correspond to the alpha orbitals and the last norb qubits correspond to the beta orbitals.

class ffsim.qiskit.OrbitalRotationJW(orbital_rotation, spin=<Spin.ALPHA_AND_BETA: 3>, label=None, validate=True, rtol=1e-05, atol=1e-08)[source]¶

Bases: Gate

Orbital rotation under the Jordan-Wigner transformation.

The orbital rotation gate has the unitary

\[\prod_{\sigma} \exp\left(\sum_{ij} \log(U)_{ij} a^\dagger_{\sigma, i} a_{\sigma, j}\right)\]

where \(U\) is a unitary matrix.

This gate assumes that qubits are ordered such that the first norb qubits correspond to the alpha orbitals and the last norb qubits correspond to the beta orbitals.
